## 🧠 What You’ll Learn

### 🔹 Level 1 – Basics

* SELECT, WHERE, ORDER BY
* Filtering (AND, OR, IN, BETWEEN)
* Aggregations (COUNT, SUM, AVG)

### 🔹 Level 2 – Intermediate

* Joins (INNER, LEFT, RIGHT)
* Subqueries
* CASE statements
* Data cleaning

### 🔹 Level 3 – Advanced

* Window functions (ROW_NUMBER, RANK, DENSE_RANK)
* Running totals
* LAG & LEAD
* Time-based analysis

### 🔹 Level 4 – Expert

* CTEs & Recursive queries
* Data modeling concepts
* Feature engineering in SQL
* Pipeline-style queries

### 🔹 Level 5 – Real-World

* Cohort analysis
* Churn detection
* Customer segmentation
* Revenue analytics
* DAU / MAU metrics
